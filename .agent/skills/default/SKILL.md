# Folder Philosophy

Prefer feature grouping over technical grouping.

Every feature owns:
- components
- services
- queries
- types
- validators
- stores
- tests

# Naming Rules

Use:
- kebab-case for folders
- PascalCase for components
- camelCase for variables
- explicit filenames

Avoid abbreviations.

# Data Integrity Rules

Quranic text is immutable.

Never:
- auto-correct ayah text
- transform Arabic content destructively
- mix tafsir with translation

# PWA Rules

Application MUST:
- work offline partially
- cache core pages
- support install prompts
- preserve reading state locally
- restore last session

# Local Storage Rules

Use local storage only for:
- bookmarks
- reading progress
- preferences
- recent searches
- AI chat history

Never store:
- sensitive credentials
- server authority data

# Design Philosophy

Visual style:
- minimal modern
- calm typography
- spacious layout
- Quran-app inspired
- distraction-free reading

Prefer readability over decoration.
