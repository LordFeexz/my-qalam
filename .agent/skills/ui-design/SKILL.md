# Reading Experience

Reading pages should:
- maximize focus
- preserve scroll position
- support font scaling
- support immersive mode
- support quick navigation

# Mobile Rules

Mobile is primary.

Design for:
- one-handed usage
- thumb reach
- fast loading on slow networks

# Search UX

Search must:
- feel instant
- support typo tolerance
- show grouped results
- support Arabic + Latin queries
- preserve recent history

# AI UI Rules

AI responses must:
- visually distinguish citations
- separate generated summaries
- show source references clearly
- avoid fake certainty styling

# Accessibility Rules

Never rely only on color.

Always:
- provide focus states
- support screen readers
- preserve semantic HTML
- support zoom up to 200%

# Empty States

Empty states should:
- educate users
- encourage exploration
- remain visually calm

# Loading States

Prefer:
- skeleton loading
- optimistic UI
- minimal spinner usage

# Navigation

Navigation hierarchy:
- Quran
- Hadith
- Sirah
- Search
- AI
- Bookmarks

Keep navigation shallow.

Avoid deep nested menus.
